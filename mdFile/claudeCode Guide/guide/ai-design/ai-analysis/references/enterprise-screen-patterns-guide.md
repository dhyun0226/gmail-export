## 화면 패턴 개요

### 패턴 분류 체계
```yaml
화면_패턴_분류:
  데이터_중심:
    - CRUD (생성/조회/수정/삭제)
    - 검색/필터링
    - 마스터-디테일
    - 트리-그리드
    
  프로세스_중심:
    - 위저드/스텝
    - 승인/워크플로우
    - 장바구니/주문
    
  시각화_중심:
    - 대시보드
    - 리포트/통계
    - 타임라인
    - 캘린더
    - 갤러리
    
  인터랙션_중심:
    - 칸반 보드
    - 채팅/메시징
    - 지도 기반
    - 비교 화면
    
  시스템_관리:
    - 설정 관리
    - 프로필 관리
    - 파일 관리
```

```javascript
const SearchComponent = {
  setup() {
    const searchQuery = ref('');
    const suggestions = ref([]);
    
    const debouncedSearch = debounce(async (query) => {
      if (query.length >= 2) {
        suggestions.value = await api.getSuggestions(query);
      }
    }, 300);
    
    watch(searchQuery, debouncedSearch);
    
    return { searchQuery, suggestions };
  }
};
```

```javascript
// Vue3 Component
const FileUpload = {
  setup() {
    const uploadFile = async (file) => {
      const formData = new FormData();
      formData.append('file', file);
      
      const response = await api.post('/files/upload', formData, {
        headers: { 'Content-Type': 'multipart/form-data' },
        onUploadProgress: (progressEvent) => {
          const progress = Math.round(
            (progressEvent.loaded * 100) / progressEvent.total
          );
          updateProgress(progress);
        }
      });
      
      return response.data;
    };
    
    return { uploadFile };
  }
};
```

## 네비게이션 패턴

### 탭 네비게이션
```javascript
// 동적 탭 관리
const useTabNavigation = () => {
  const tabs = ref([
    { id: 1, title: '홈', component: 'HomeTab', closable: false },
    { id: 2, title: '상품관리', component: 'ProductTab', closable: true }
  ]);
  const activeTab = ref(1);
  
  const addTab = (tab) => {
    tabs.value.push({ ...tab, id: Date.now() });
  };
  
  const closeTab = (tabId) => {
    const index = tabs.value.findIndex(t => t.id === tabId);
    if (index > -1) {
      tabs.value.splice(index, 1);
      if (activeTab.value === tabId) {
        activeTab.value = tabs.value[0]?.id;
      }
    }
  };
  
  return { tabs, activeTab, addTab, closeTab };
};
```
