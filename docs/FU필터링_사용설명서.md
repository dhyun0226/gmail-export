# FU 필터링 프로그램 사용설명서


---


## 이 프로그램은 뭔가요?

원본 엑셀 파일에서 **FU 열**을 기준으로,
지정한 코드에 해당하는 행만 골라서 **새 엑셀 파일로 만들어주는** 프로그램입니다.

엑셀을 직접 열지 않아도 되기 때문에 **파일이 아무리 커도** (80만행 이상) 처리할 수 있습니다.


---


## 준비물

| 순서 | 파일 | 설명 |
|------|------|------|
| 1 | **FU필터링.exe** | 이 프로그램 (전달받은 파일) |
| 2 | **원본.xlsx** | FU 열이 들어있는 대용량 원본 엑셀 |
| 3 | **필터.xlsx** | A열에 필터링할 코드 목록이 적힌 엑셀 |

> **중요:** 파일명이 정확히 `원본.xlsx`, `필터.xlsx` 이어야 합니다!


---


## 사용 방법 (단계별)


### 1단계: 폴더 만들기

바탕화면이나 원하는 곳에 **새 폴더**를 하나 만듭니다.

폴더 이름은 아무거나 상관없습니다. (예: "FU작업", "필터링" 등)


### 2단계: 파일 3개를 폴더에 넣기

만든 폴더 안에 아래 3개 파일을 넣어주세요:

```
📁 FU작업 (폴더)
 ├── FU필터링.exe      ← 프로그램
 ├── 원본.xlsx         ← 원본 엑셀 (파일명을 "원본.xlsx"로 변경!)
 └── 필터.xlsx         ← 필터링 코드 엑셀 (파일명을 "필터.xlsx"로 변경!)
```

⚠️ **파일명 변경 방법:**
- 원본 엑셀 파일에서 마우스 우클릭 → "이름 바꾸기" → `원본.xlsx` 로 입력
- 필터링 엑셀도 같은 방법으로 `필터.xlsx` 로 변경


### 3단계: 프로그램 실행

`FU필터링.exe` 파일을 **더블클릭**합니다.

검은색 창(명령 프롬프트)이 열리면서 아래와 같이 진행됩니다:

```
========================================
       FU 필터링 프로그램 v1.0
========================================

[1/4] 필터링 파일 읽는 중...
  -> 필터링 코드 150개 로드 완료

[2/4] 원본 파일 읽는 중... (파일이 크면 시간이 걸립니다)
  -> 원본 데이터 806,678행 로드 완료
  -> FU 열 위치: 5번째 열

[3/4] 필터링 중...
  -> 100,000행 처리 중... (매칭 1,234건)
  -> 200,000행 처리 중... (매칭 2,567건)
  ...
  -> 필터링 완료: 총 806,678행 중 12,345건 매칭

[4/4] 결과 파일 생성 중...

========================================
  완료! "결과.xlsx" 파일이 생성되었습니다.
  총 12,345건의 데이터가 추출되었습니다.
========================================

아무 키나 누르면 종료됩니다...
```


### 4단계: 결과 확인

같은 폴더에 **결과.xlsx** 파일이 새로 생깁니다.

```
📁 FU작업 (폴더)
 ├── FU필터링.exe
 ├── 원본.xlsx
 ├── 필터.xlsx
 └── 결과.xlsx         ← ✅ 새로 생긴 결과 파일!
```

`결과.xlsx`를 열면 필터링된 데이터만 들어있습니다.


---


## 원본 엑셀 조건

- 첫 번째 행(1행)이 **헤더**여야 합니다
- 헤더 중에 **"FU"** 라는 이름의 열이 있어야 합니다
- 헤더는 결과 파일에도 그대로 포함됩니다

예시:
| ... | FU | ... | ... |
|-----|-----|-----|-----|
| ... | ABC123 | ... | ... |
| ... | DEF456 | ... | ... |


## 필터링 엑셀 조건

- **A열**에 필터링할 코드를 넣어주세요
- **헤더 없이** 1행부터 바로 코드를 적어주세요

예시:
| A열 |
|-----|
| ABC123 |
| GHI789 |
| JKL012 |


---


## 처리 시간 안내

| 원본 크기 | 예상 시간 |
|-----------|-----------|
| 10만행 이하 | 수 초 |
| 10만~50만행 | 1~3분 |
| 50만~100만행 | 3~10분 |

> 파일이 클수록 시간이 오래 걸립니다. **검은 창을 닫지 말고 기다려주세요!**


---


## 오류가 나는 경우

| 오류 메시지 | 원인 | 해결 방법 |
|------------|------|-----------|
| "원본.xlsx" 파일을 찾을 수 없습니다 | 파일명이 다름 | 원본 엑셀 파일명을 `원본.xlsx`로 변경 |
| "필터.xlsx" 파일을 찾을 수 없습니다 | 파일명이 다름 | 필터 엑셀 파일명을 `필터.xlsx`로 변경 |
| "FU" 열을 찾을 수 없습니다 | 헤더에 FU가 없음 | 원본 엑셀의 1행에 "FU" 열이 있는지 확인 |
| 프로그램이 바로 꺼짐 | 파일이 같은 폴더에 없음 | 3개 파일이 같은 폴더에 있는지 확인 |


---


## 주의사항

1. **파일명을 꼭 맞춰주세요** — `원본.xlsx`, `필터.xlsx` (대소문자 무관)
2. **3개 파일이 같은 폴더에 있어야 합니다**
3. **처리 중 검은 창을 닫지 마세요** — 강제 종료됩니다
4. **기존 결과.xlsx가 있으면 덮어씁니다** — 이전 결과는 미리 다른 곳에 저장하세요
5. **원본 파일은 절대 수정되지 않습니다** — 안심하세요
