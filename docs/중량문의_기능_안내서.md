# 중량문의 조회 기능 안내서


---


## 이 기능이 뭔가요?

메일함에 있는 **중량문의 메일**을 자동으로 찾아서,
메일 하나하나 열어볼 필요 없이 **한 번에 데이터를 뽑아주는 기능**입니다.


---


## 어디서 쓰나요?

사이드바 메뉴에 **"중량문의"** 가 새로 추가되었습니다.

```
마일스톤
Cargo List
KPI
중량문의    ← 여기!
환율
```


---


## 사용 방법

### 1단계: 조회 버튼 클릭

페이지에 들어가면 **"중량문의 메일 조회"** 버튼이 있습니다.
클릭하면 **최근 6개월치** 중량문의 메일을 자동으로 검색합니다.

> 메일이 많으면 시간이 좀 걸릴 수 있어요. (조회 시간이 화면에 표시됩니다)

### 2단계: 결과 확인

조회가 끝나면 아래와 같은 테이블이 나옵니다:

| 메일 일자 | HB 번호 | MB 번호 | 적하중량 | 순중량 | 적하중량 진행 |
|-----------|---------|---------|----------|--------|:---:|
| 2026-03-10 | 4812435711 | 297-71116065 | 1 KG | 1.04 KG | O |
| 2026-03-08 | 4812435600 | 297-71115900 | 2.5 KG | 2.8 KG | X |

### 3단계: Excel 다운로드

테이블 오른쪽 위에 **"Excel 다운로드"** 버튼을 누르면
위 데이터가 그대로 엑셀 파일로 저장됩니다.


---


## 각 항목 설명

| 항목 | 어디서 가져오나 | 설명 |
|------|----------------|------|
| **메일 일자** | 메일 발송일 | 중량문의 메일이 보내진 날짜 |
| **HB 번호** | 메일 제목 | `HB: 4812435711` 형태에서 추출 |
| **MB 번호** | 메일 제목 | `MB: 297-71116065` 형태에서 추출 |
| **적하중량** | 메일 본문 | `적하중량 : 1 KG` 형태에서 추출 |
| **순중량** | 메일 본문 | `순중량 : 1.04 KG` 형태에서 추출 |
| **적하중량 진행** | RE: 답변 메일 | 답변에 "적하 중량에 맞추어 진행" 문구가 있으면 **O**, 없으면 **X** |


---


## 적하중량 진행 (O/X) 판별 방식

중량문의 메일에 답변(RE:)이 달리면, 그 답변 내용을 자동으로 확인합니다.

- 답변에 **"적하 중량"** + **"진행"** 이라는 문구가 포함되어 있으면 → **O**
- 답변이 없거나 해당 문구가 없으면 → **X**

### 예시

답변 내용이 이런 경우 → **O**:
> 적하 중량에 맞추어 진행 부탁드립니다.

답변 내용이 이런 경우 → **X**:
> 순중량 기준으로 정정해주세요.

답변이 아예 없는 경우 → **X**


---


## 중복은 어떻게 처리하나요?

같은 건에 대해 원본 메일 + RE: 답변이 여러 개 있어도
**스레드(대화 묶음) 단위로 1건만** 표시됩니다.

예를 들어:
- 중량문의 원본 메일 1통
- RE: 답변 2통

이 3통은 같은 스레드이므로 결과에는 **1건**으로 나옵니다.


---


## 검색 대상 메일

- 제목에 **중량문의**, **[중량문의]**, ***중량문의*** 가 포함된 메일
- **최근 6개월** 범위
- 본문에 적하중량 또는 순중량 데이터가 있는 것만 결과에 포함
