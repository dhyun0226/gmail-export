# KPI 시스템 수정사항 안내 (2026.03.12)


---


## 수정 1. Delay Reason 사유 넣는 조건 변경 (3번)

### 변경 전
- 수리일(H열)이 접수일(G열)보다 **하루 이상 늦으면** → "Delay in bringing cargo..." 사유가 들어감
- 너무 단순한 조건이라 정확하지 않았음

### 변경 후
아래 **두 가지 조건을 동시에 충족**해야 사유가 들어갑니다:

1. **G열(Submitted to Customs)의 시간이 F열(Warehouse)보다 빠른 경우**
   - 통관접수가 창고반입보다 먼저 이루어졌다는 뜻
   - 즉, 화물이 아직 보세창고에 안 들어왔는데 통관접수부터 한 상황

2. **F열(Warehouse)과 H열(Custom Clearance)의 시간 차이가 2분 이내인 경우**
   - 창고에 반입되자마자 바로 수리가 됐다는 뜻
   - 이미 통관접수는 되어있어서 창고만 들어오면 바로 처리된 것

### 의미
> 화물이 보세창고에 늦게 도착해서 지연이 발생한 케이스를 정확히 잡아냅니다.
> 항공사의 화물 운송 지연으로 인한 것이므로 **Uncontrollable** 사유입니다.

## 수정 2. Gross=N & Net=N인 BL 하이라이트 (4번)

### Gross(P열)와 Net(Q열)이 뭔가요?

| 열 | 의미 | Y일 때 | N일 때 |
|---|------|--------|--------|
| **P (Gross)** | KPI 시간을 지켰는가? | 지켰음 (Diff time ≤ 0) | 못 지켰음 (Diff time > 0) |
| **Q (Net)** | 실질적으로 우리 책임인가? | 지켰거나, 못 지켰지만 불가항력(Uncontrollable) | **못 지켰고 우리 책임** |

### N/N이란?

**Gross=N, Net=N** 이란:
- KPI 시간을 초과했고 (Diff time > 0)
- Uncontrollable 사유도 없다

즉, **우리 쪽에서 통제 가능했는데 시간을 넘긴 건**입니다.
→ 가장 주의가 필요한 건이므로 **진한 빨간색으로 강조 표시**됩니다.

### 화면 표시

| 상태 | 표시 |
|------|------|
| Gross=N & Net=N | 진한 빨간 배경 + 굵은 빨간 왼쪽 테두리 |
| Net=N만 (Gross=Y) | 연한 빨간 배경 + 빨간 왼쪽 테두리 |
| 그 외 | 일반 표시 |

## 수정 3. "Out of broker's office hours" 조건 수정 (5번)

### 변경 전
- **DHL 메일 수신 시간** 기준으로 09시 이전 / 18시 이후면 사유 적용
- 문제: 메일이 언제 왔는지는 업무시간 외 지연과 직접 관련이 없음

### 변경 후
- **E열 (Dest.Arrival = 하기신고수리일시)** 기준으로 09시 이전 / 18시 이후면 사유 적용

### 의미
> 화물이 도착한 시간이 업무시간(09~18시) 밖이면
> 처리가 늦어질 수밖에 없으므로 **Uncontrollable** 사유로 인정합니다.
>
> 예) Dest.Arrival이 오후 7시 → 당일 처리 불가 → 다음날 처리 → Diff time 초과

## 참고: Delay Reason 적용 전체 흐름

사유는 **Diff time이 0을 초과하는 건에 대해서만** 아래 순서로 적용됩니다.
(Diff time ≤ 0이면 KPI를 준수한 것이므로 사유 자체가 불필요)

```
Diff time > 0인 건
  │
  ├─ 1순위: 키워드 매칭 (검사, RWA, 세관장 등)
  │         → 사유(키워드) 파일의 특이사항에서 매칭
  │
  ├─ 2순위: DHL Doc > Dest.Arrival
  │         → Pre-alert missing by carriers
  │
  ├─ 3순위: G < F 이고 F ≈ H (2분 이내)
  │         → Delay in bringing cargo into bonded warehouse
  │
  ├─ 4순위: Dest.Arrival이 업무시간 외 (09시 전 / 18시 후)
  │         → Out of broker's office hours
  │
  └─ 해당 없음 → 사유 없음 (Controllable)
```
